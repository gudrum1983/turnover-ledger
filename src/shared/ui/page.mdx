# Turnover Ledger UI

Storybook для общих UI-компонентов и CSS-утилит проекта.

Здесь собраны базовые элементы из `src/shared/ui`: кнопки, поля, списки, модалки, layout-токены, типографика и иконки. Истории нужны как живая документация: проверить состояния компонента, посмотреть доступные props и быстро найти пример использования.

## Как читать

- **Песочница** — интерактивная история с controls для проверки props.
- **По умолчанию** — базовый сценарий компонента.
- **❗ Истории** — некорректные или пограничные состояния, которые полезно видеть при разработке.

## Оглавление

### Foundation

- [Layout](?path=/docs/ui-утилиты-layout--docs) — `Container`, breakpoints, container tokens и формула ширины.
- [Typography](?path=/docs/ui-утилиты-typography--docs) — текстовые классы и шрифтовые токены.
- [Text](?path=/docs/ui-утилиты-text--docs) — утилитарные классы для текста.

### Actions

- [ButtonBase](?path=/docs/ui-компоненты-buttonbase--docs) — базовая кнопка с вариантами, цветами, размерами и слотами иконок.
- [ButtonDropdown](?path=/docs/ui-компоненты-buttondropdown--docs) — кнопка с выпадающим списком и одиночным выбором.
- [ButtonGroup](?path=/docs/ui-компоненты-buttongroup--docs) — группа кнопок с одиночным выбором и клавиатурной навигацией.

### Links

- [LinkBase](?path=/docs/ui-компоненты-linkbase--docs) — внутренняя ссылка на `RouterLink`.
- [LinkExternal](?path=/docs/ui-компоненты-linkexternal--docs) — внешняя ссылка, которая открывается в новом окне.
- [LinkEmail](?path=/docs/ui-компоненты-linkemail--docs) — почтовая ссылка с `mailto:`.

### Inputs*

\* Поля ввода и связанные с ними read-only компоненты для форм.

- [FieldBase](?path=/docs/ui-компоненты-fieldbase--docs) — базовое текстовое поле с очисткой, debounce и `blur`.
- [FieldCounter](?path=/docs/ui-компоненты-fieldcounter--docs) — текстовое поле со счётчиком длины.
- [FieldDate](?path=/docs/ui-компоненты-fielddate--docs) — выбор даты через `@vuepic/vue-datepicker`.
- [FieldDigit](?path=/docs/ui-компоненты-fielddigit--docs) — поле только для цифр.
- [FieldMoney](?path=/docs/ui-компоненты-fieldmoney--docs) — денежное поле с локальным десятичным разделителем.
- [DisplayField](?path=/docs/ui-компоненты-displayfield--docs) — read-only поле для показа значения рядом с обычными инпутами.

### Selection

- [ListBox](?path=/docs/ui-компоненты-listbox--docs) — список опций с одиночным выбором и избранными значениями.

### Layout And Surfaces

- [DividerBase](?path=/docs/ui-компоненты-dividerbase--docs) — горизонтальный или вертикальный разделитель.
- [DividerToggle](?path=/docs/ui-компоненты-dividertoggle--docs) — разделитель с кнопкой раскрытия.
- [PaperBase](?path=/docs/ui-компоненты-paperbase--docs) — базовая поверхность с отступами, фоном и тенью.

### Feedback And Overlays

- [LoaderBase](?path=/docs/ui-компоненты-loaderbase--docs) — базовый точечный индикатор загрузки с размерами для компактных и обычных сценариев.
- [ModalBase](?path=/docs/ui-компоненты-modalbase--docs) — базовая модалка с overlay, teleport и блокировкой скролла.
- [DialogConfirm](?path=/docs/ui-компоненты-dialogconfirm--docs) — готовый диалог подтверждения или удаления.
- [TagBase](?path=/docs/ui-компоненты-tagbase--docs) — компактная служебная метка.

### Assets

- [Icons](?path=/docs/ui-компоненты-icons--docs) — обзор SVG-иконок из `src/shared/ui/icons`.
